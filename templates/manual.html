Manual de Usuario

Guía práctica para usar la aplicación

1. Introducción

Este manual tiene como objetivo ayudarte a comprender y utilizar todas las funciones de la aplicación de manera eficiente. Sigue las instrucciones paso a paso para aprovechar al máximo la herramienta.

2. Requisitos del Sistema

  • Sistema operativo: Windows, macOS, o Linux.
  • Navegador recomendado: Google Chrome, Mozilla Firefox y teléfonos móviles.
  • Conexión a internet estable.

3. Instrucciones de Uso

  1. Descargar e instalar: Accede al sitio oficial y descarga la aplicación. Sigue el proceso de instalación indicado.
  2. Iniciar sesión: Abre la aplicación y usa tus credenciales para ingresar. Si no tienes una cuenta, regístrate primero.
  3. Navegar: Usa el menú principal para acceder a las funciones disponibles.
  4. Realizar tareas: Selecciona la acción deseada (SMS, Calendario o Crear Contactos) desde el panel de opciones.

4. Soporte Técnico

Si encuentras algún problema, puedes contactar a nuestro equipo de soporte enviando un correo a crjfjuyg@gmail.com o llamando al +56958983617.

5. Preguntas Frecuentes

  • ¿Cómo restablezco mi contraseña? Usa la opción "¿Olvidaste tu contraseña?" en la página de inicio de sesión.
  • ¿Puedo usar la aplicación sin conexión? Algunas funciones están disponibles sin conexión, pero otras requieren acceso a internet.
